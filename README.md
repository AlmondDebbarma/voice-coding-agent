# Voice Coding Agent

## Overview
Voice Coding Agent is a project that aims to streamline coding tasks through voice commands. It is designed to improve productivity by allowing developers to interact with coding environments using natural language.

## Features
- Voice recognition for coding commands
- Integration with popular coding tools
- Customizable commands and responses

## Setup Instructions

### Prerequisites
- Python 3.8 or higher
- Necessary libraries (see requirements.txt)

### Environment Variables
To set up the environment variables, create a `.env` file in the root directory of the project and include the following:
```
GROQ_KEY=your_groq_key_here
```
Replace `your_groq_key_here` with your actual GROQ key.

### Running the Project
1. Clone the repository:
```bash
git clone https://github.com/AlmondDebbarma/voice-coding-agent.git
cd voice-coding-agent
```

2. Install the required dependencies:
```bash
pip install -r requirements.txt
```

3. Run the agent using the following command:
```bash
python statbot.py
```

4. Start using voice commands to interact with your coding tasks!
